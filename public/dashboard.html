Bot Setup
The Cordfol bot enables enhanced server-side role verification. Add it to any server you manage.
⚡ How the dual-bot architecture works
👤 User OAuth Bot
Used when you log in. Reads your own guild membership and roles. Only sees data about you, nobody else.
🤖 Server Bot
Sits in your server with READ_MEMBERS permission. When a user logs into Cordfol, the bot confirms their role — adding a second layer of proof.
Add the bot to a server
You must have Manage Server permission in the target Discord server.
